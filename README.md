基于springboot酒店宾馆管理系统


项目代码已收录公众号【java项目源码】，需要请自行关注一下公众号并下载源码

![公众号二维码](./运行截图/wechat.png)

主要分三种角色：管理者/工作人员/前台人员。

主要功能有：
客房：客房标准、房间信息；
订单：入住订单；
员工：员工信息；
事务：事务信息；
停车：车位信息；
财务：财务信息；
历史：订单历史、车库历史；
会员：会员信息；

1. 后端：SpringBoot

2. 前端：html+layui+jQuery
